# CrossBrowserSuite TestNG XML

This is a TestNG XML configuration file for running automated tests on multiple browsers.

## Overview

- The suite is named `CrossBrowserSuite`.
- It contains two tests to run the same test class on different browsers.
- The class run in both tests is `com.testautomation.login.NegativeLoginTests`.

## Tests description

- **ChromeTest:** Runs tests with the browser parameter set to Chrome.
- **FireFoxTest:** Runs tests with the browser parameter set to Firefox.
