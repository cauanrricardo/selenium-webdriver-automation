This module explores the relationship between **HTML structure**, the **Document Object Model (DOM)**, and **Selenium WebDriver automation**. It focuses on how browsers interpret HTML elements and how Selenium interacts with them through locators.

### Objectives

- Understand how **HTML tags**, **attributes**, and the **DOM structure** define web elements.  
- Learn to **inspect and analyze web elements** using browser developer tools.  
- Identify elements through different **locator strategies** such as **ID**, **name**, **class**, **tag**, **link text**, **CSS selector**, and **XPath**.  
- Practice using **explicit waits** (`WebDriverWait`) and element state methods (`isDisplayed`, `isEnabled`, etc.).  
- Validate **messages, buttons, and interactions** on dynamic web pages.  

All exercises were developed and executed using **IntelliJ IDEA**, **Firefox**, and **Linux**.
