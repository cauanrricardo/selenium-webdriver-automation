This module introduces the TestNG testing framework and its integration with Selenium WebDriver. It focuses on the use of annotations, assertions, and test structure organization to build maintainable and reusable automated tests.

## Objectives

- Understand the role of TestNG in test automation.
- Learn the main TestNG annotations such as `@BeforeMethod`, `@AfterMethod`, and `@Test`.
- Implement test setup and teardown logic for efficient browser management.
- Use Assert methods to validate expected outcomes.
- Execute and generate HTML reports using the Maven Surefire plugin.

*All exercises were developed and executed using IntelliJ IDEA, Firefox, TestNG, and Maven on Linux.*
