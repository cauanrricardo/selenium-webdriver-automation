## Overview
contains automated test suites for login functionality testing using TestNG framework.
## Test Suites
### 1. Smoke Test Suite `(smoke-test-suite.xml)`
A lightweight test suite designed for quick validation of critical functionality.
Purpose: Rapid smoke testing to verify basic login functionality
##### Test Coverage:
- Positive Login Test
### 2. Full Test Suite (full-test-suite.xml)
A comprehensive test suite covering both positive and negative login scenarios.
Purpose: Complete regression testing of login functionality
##### Test Coverage:
- Positive Login Test
- Negative Login Test
